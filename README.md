# 📊 ASX Financial Report Intelligence Platform

An AI-powered financial analysis platform that automatically extracts insights from ASX company annual reports using Large Language Models.

🔗 **Live Demo:** https://asx-financial-analyser.streamlit.app/

---

## 🚀 What This Project Does

Australian financial analysts spend hours manually reading through hundreds of pages of ASX annual reports to extract key metrics, identify risks, and summarise performance. This platform automates that entire process in seconds.

Upload any ASX company annual report PDF and instantly get:

- 📈 Financial metrics — Revenue, Profit, EBITDA, Dividend
- ⚠️ Key business risks ranked by priority
- 🚀 Growth opportunities identified automatically
- 📝 Executive summary generated by AI
- 🔮 Future outlook extracted from management commentary
- 📊 Interactive risk visualisation charts
- 🆚 Side by side comparison of two companies

## 💡 Key Features

### Single Company Analysis
Upload any ASX annual report and get a complete financial intelligence report in seconds.

### Multi Company Comparison
Upload two reports simultaneously and compare companies side by side with visual charts.

### Download Analysis
Export the complete analysis as a structured JSON file for further processing.

## 🔧 How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/Shubham-Reddy/asx-financial-analyser.git
cd asx-financial-analyser
```

2. Create conda environment
```bash
conda create -n asx_env python=3.11
conda activate asx_env
conda install -c conda-forge streamlit pdfplumber pandas plotly python-dotenv
conda install -c conda-forge groq
```

3. Add your Groq API key
```bash
# Create .env file
echo "GROQ_API_KEY=your_groq_api_key_here" > .env
```

Get your free Groq API key at: https://console.groq.com

4. Run the app
```bash
streamlit run app.py
```

As a data science student studying the Australian job market, I noticed that financial analysts spend significant time manually processing ASX annual reports.
